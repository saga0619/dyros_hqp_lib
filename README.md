## How to ...
### prerequisites
 * Eigen3
 * [RBDL](https://github.com/saga0619/rbdl-orb)
 * [qpOASES](https://github.com/saga0619/qpOASES)
 * [qpSWIFT](https://github.com/qpSWIFT/qpSWIFT)(wip)

### How to install
```sh
mkdir build
cd build 
cmake ..
make
sudo make install
```
